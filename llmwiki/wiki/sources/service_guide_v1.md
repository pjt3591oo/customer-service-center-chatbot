회원가입 및 로그인 가이드
분류: 회원 | ID: guide-001
1. 메인 화면 우측 상단의 '회원가입' 버튼을 클릭합니다.
2. 이메일, 비밀번호, 휴대폰 번호를 입력하고 본인인증을 진행합니다.
3. 약관 동의 후 '가입완료' 버튼을 클릭하면 가입이 완료됩니다.
4. 가입 후 입력한 이메일과 비밀번호로 로그인할 수 있으며, 카카오/구글 간편로그인도 지원합니다.
5. 비밀번호는 영문, 숫자, 특수문자를 포함하여 8자 이상으로 설정해야 합니다.
관련 QnA: qna-012, qna-013
주문 및 결제 진행 방법
분류: 주문/결제 | ID: guide-002
1. 원하는 상품의 상세 페이지에서 옵션(사이즈, 색상 등)을 선택한 후 '구매하기' 또는 '장바구니'를
클릭합니다.
2. 장바구니에서 여러 상품을 한 번에 주문할 수 있으며, '주문하기' 버튼을 클릭해 결제 화면으로
이동합니다.
3. 배송지 정보를 입력하거나 기존에 저장된 배송지를 선택합니다.
4. 사용 가능한 쿠폰과 적립금을 적용합니다.
5. 결제 수단(카드, 간편결제, 무통장입금 등)을 선택하고 '결제하기'를 클릭하면 주문이 완료됩니다.
6. 주문 완료 후에는 마이페이지 > 주문내역에서 진행 상태를 확인할 수 있습니다.
관련 QnA: qna-006, qna-007, qna-008
주문 취소 절차
분류: 주문/결제 | ID: guide-003
1. 마이페이지 > 주문내역으로 이동합니다.
2. 취소하고자 하는 주문 항목 옆의 '주문취소' 버튼을 클릭합니다.
3. 취소 사유를 선택하고 '취소 신청'을 완료합니다.
4. 주문 상태가 '상품준비중'일 경우에만 즉시 취소가 가능하며, '배송중' 또는 '배송완료' 상태에서는
취소가 불가능하고 반품 절차를 이용해야 합니다.
5. 취소 완료 시 결제 수단에 따라 1~5 영업일 내에 환불이 처리됩니다.
관련 QnA: qna-005, qna-010



반품/교환 신청 절차
분류: 반품/교환/환불 | ID: guide-004
1. 마이페이지 > 주문내역에서 반품/교환하고자 하는 상품을 선택합니다.
2. '반품/교환 신청' 버튼을 클릭한 후, 사유(단순변심, 파손/불량, 오배송 등)를 선택합니다.
3. 파손/불량의 경우 상품 사진을 첨부하면 처리가 더 빠르게 진행됩니다.
4. 신청이 완료되면 택배사가 방문하여 상품을 수거합니다. (단순변심은 고객이 직접 발송해야 할
수도 있습니다.)
5. 상품이 물류센터에 도착 후 검수가 완료되면 교환 상품이 재발송되거나 환불이 진행됩니다.
6. 신청 가능 기간은 상품 수령일로부터 7일 이내입니다.
관련 QnA: qna-009, qna-010, qna-011
배송 조회 방법
분류: 배송 | ID: guide-005
1. 마이페이지 > 주문내역으로 이동합니다.
2. 배송 중인 주문 항목 옆의 '배송조회' 버튼을 클릭합니다.
3. 연동된 택배사의 실시간 배송 추적 페이지로 이동하여 현재 위치 및 예상 도착 시간을 확인할 수
있습니다.
4. 운송장 번호는 상품 출고 후 약 1시간 이내에 자동으로 등록됩니다.
5. 배송 상태는 '상품준비중 → 배송중 → 배송완료'의 순서로 업데이트됩니다.
관련 QnA: qna-001, qna-002
쿠폰 및 적립금 사용 방법
분류: 프로모션 | ID: guide-006
1. 마이페이지 > 쿠폰함에서 보유 중인 쿠폰 목록과 사용 기한을 확인할 수 있습니다.
2. 결제 화면의 '쿠폰 적용' 영역에서 사용할 쿠폰을 선택하면 할인 금액이 자동 반영됩니다.
3. 적립금은 결제 화면의 '적립금 사용' 영역에서 직접 사용할 금액을 입력합니다. (최소 1,000원
단위)
4. 쿠폰과 적립금은 동시에 사용할 수 있으나, 일부 프로모션 상품은 중복 사용이 제한될 수 있습니다.
5. 적립금은 구매 확정 후 상품 금액의 일정 비율로 자동 적립됩니다.
관련 QnA: qna-008, qna-014



회원 탈퇴 절차
분류: 회원 | ID: guide-007
1. 마이페이지 > 설정 > 회원정보 관리로 이동합니다.
2. 화면 하단의 '회원탈퇴' 버튼을 클릭합니다.
3. 탈퇴 사유를 선택하고 안내사항(보유 적립금/쿠폰 소멸, 주문내역 접근 제한 등)에 동의합니다.
4. 진행 중인 주문이 있거나 미사용 적립금/쿠폰이 남아있는 경우, 이를 먼저 처리해야 탈퇴가
가능합니다.
5. 탈퇴 완료 시 즉시 계정이 비활성화되며, 동일 이메일로는 30일 이후 재가입이 가능합니다.
관련 QnA: qna-013, qna-014
영수증 및 세금계산서 발급
분류: 기타 | ID: guide-008
1. 마이페이지 > 주문내역에서 발급받고자 하는 주문을 선택합니다.
2. '영수증 출력' 버튼을 클릭하면 현금영수증 또는 신용카드 매출전표를 PDF로 다운로드할 수
있습니다.
3. 사업자 회원의 경우 세금계산서 발급을 위해 고객센터에 사업자등록번호와 함께 요청해야 합니다.
4. 세금계산서는 신청일로부터 영업일 기준 3일 이내에 이메일로 발송됩니다.
관련 QnA: qna-016
